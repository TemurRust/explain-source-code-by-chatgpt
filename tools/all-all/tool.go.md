3. `SubCommand`结构体：表示一个命令行子命令。它包含了子命令的名称、帮助信息、运行方法等。

4. `commandLineError`结构体：表示命令行解析错误。它包含了错误消息以及命令行参数的使用方法。

4. `Run`函数：用于运行命令行工具。
