newCmdDiff函数是用来创建Kubeadm升级过程中差异比较的命令对象。它会设置命令的名称、描述和运行函数。

validateManifestsPath函数是用来验证指定的配置文件路径是否存在，如果不存在则返回错误。

runDiff函数是实际执行差异比较的函数。它会根据指定的配置文件路径和标志来进行差异比较，并输出结果。
