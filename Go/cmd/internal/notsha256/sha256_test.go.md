# File: sha256_test.go

sha256_test.go文件是Go语言标准库中SHA256哈希算法的测试文件。其中包含了对SHA256哈希算法的多组测试用例，用于验证SHA256函数的正确性和准确性。

该测试文件主要包含以下内容：

1. 导入测试所需的包和库。

2. 定义了一个测试函数TestSHA256，用于测试SHA256函数的正确性。该函数中包含了多组测试用例，通过比较SHA256哈希值的结果来验证函数的正确性。

3. 定义了一个例子函数ExampleSum, 用于演示如何使用SHA256哈希算法计算一个数据的散列值。

通过这个测试文件，我们可以充分验证SHA256哈希算法的正确性和准确性，确保在使用SHA256函数时数据可以得到正确的哈希值。
